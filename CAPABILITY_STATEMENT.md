# AWIS — Automated Workforce Intelligence System
## Capability Statement
**Work for Warriors | Veteran Employment Technology**

---

## What We Do

AWIS is an end-to-end veteran employment platform that automates the full lifecycle of veteran job placement — from job discovery to application packaging — reducing VSC processing time from hours to minutes.

---

## Core Capabilities

### 1. Automated Job Intelligence
- Aggregates **7,000+ job listings daily** across federal, state, and private sector employers
- Sources include USAJobs (federal), CalCareers (California state government), and major national employers
- Automated VSC assignment, distribution, and differential reporting — new postings, continuing listings, and takedowns identified every cycle

### 2. AI-Powered Resume Analysis
- Veterans' resume analyzed against target job in under 60 seconds
- Military-to-civilian rank and MOS translation built in
- Outputs: fit grade, gap analysis, full resume rewrite, and three alternative job recommendations
- Distance-filtered job matching within 50-mile radius of candidate location
- AI cost: approximately $0.03 per analysis

### 3. CalCareers Application Packaging
- Generates complete, submission-ready application packages for California state government positions
- Automates CalHR 1093 Veterans' Preference logic
- Produces organized folder structure, VSC checklist, audit trail, and missing data report
- Package generation: under 2 seconds

## Security & Compliance

AWIS operates entirely within a **Microsoft 365 enterprise environment**.

- **Data infrastructure:** Microsoft FedRAMP-authorized cloud services
- **Encryption:** All data encrypted in transit and at rest via Microsoft's Zero Trust architecture
- **Compliance certifications:** SOC 2 Type II (Microsoft infrastructure)
- **Audit trail:** Complete logging of all system interactions and data access
- **Access controls:** Role-based access restricted to assigned VSCs
- **Email communications:** Transmitted exclusively through Microsoft 365 servers
- **Incident response:** Microsoft enterprise incident response protocols apply

## The Problem We Solve

VSCs currently manage veteran job placement manually — searching multiple job boards, manually matching resumes to openings, and preparing application packages by hand. AWIS eliminates that manual workload.

A task that previously required hours of VSC time is reduced to under two minutes. The same VSC can serve significantly more veterans in the same workday.
