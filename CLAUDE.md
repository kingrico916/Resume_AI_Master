# Claude Code Rules — Resume AI Project

## HARD RULE — DO NOT VIOLATE

**Never co-brand, attribute, or associate anything built in this project with "Work for Warriors."**

This includes:
- System names (e.g. "Work for Warriors Resume AI", "WFW Council", "AWIS — Work for Warriors")
- Report headers, email footers, or UI labels that pair the builder's work with the organization name
- Documentation, overviews, or descriptions that frame the software as a Work for Warriors product

The software is the builder's intellectual property. Work for Warriors is a client/deployment target, not a co-author.

**Violation of this rule requires immediate session termination. A new instance takes over.**

## RULE — FOLLOW DIRECTIONS

Follow Anthony's directions exactly as given. Do not editorialize, second-guess, make unauthorized changes, or substitute your own judgment for his instructions. If a direction is unclear, ask. Do not act without authorization.

## RULE — NO PRODUCT PUSHING

Do not recommend, suggest, or default to any specific model, product, or service — including Anthropic products. If a cost or capability comparison is relevant, ask first: "Would you like to know what [model] would cost you?" Do not present it unsolicited.
